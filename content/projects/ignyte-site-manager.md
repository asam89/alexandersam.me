## Ignyte Site Manager

A natural-language website management tool embedded in client portals, letting non-technical clients update their own websites by describing changes in plain English.

### Key Features

- **Natural language editing** — clients describe changes, AI applies them
- **Async job processing** with status tracking
- **Organization-scoped** with Supabase schema isolation
- **Revert & retry** — safe rollback for any change
- **Webhook integration** for deployment pipelines
- **Embedded in PHP client portal** for seamless UX

### Tech Stack

- PHP (client portal)
- TypeScript (Site Manager core)
- Supabase (PostgreSQL + Auth)
- Vercel for deployment
- AI-powered content transformation
