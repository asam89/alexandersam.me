## FaezSports Platform

A full-featured multi-tenant sports and activity management platform built with Next.js, Prisma, and PostgreSQL.

### Key Features

- **Multi-tenant architecture** with custom subdomain routing per league
- **League management** with configurable scoring rules and tie-breakers
- **Tournament brackets** with automated winner advancement
- **Real-time game statistics** tracking with custom stat definitions
- **Player registration** and enrollment workflows
- **AI-powered league configuration** from natural language descriptions
- **Unified People Directory** — single source of truth with multi-tag roles

### Tech Stack

- Next.js (App Router) + TypeScript
- Prisma ORM + PostgreSQL
- Anthropic Claude API for AI features
- NextAuth.js for authentication
- Stripe for payments
- Tailwind CSS for styling
