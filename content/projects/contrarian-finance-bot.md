## Contrarian Finance Bot

A sentiment-inverting finance analysis bot that challenges conventional market wisdom.

### Key Features

- **Contrarian analysis** — inverts market sentiment for alternative perspectives
- **Ride the Wave picks** with meaningful analysis
- **Momentum vs contrarian strategy toggle** — switch between approaches
- **Telegram integration** for real-time alerts and analysis delivery
- **Deployed on OCI Ampere** for cost-efficient always-on operation

### Tech Stack

- Python
- Telegram Bot API
- AI-powered sentiment analysis
- Oracle Cloud Infrastructure (OCI Ampere)
