## IgnyteDev Bot

AI-powered development bot that turns natural language into deployed code.

### Key Features

- **Code generation** via Gemini 2.5 Flash from plain English descriptions
- **Git workflow automation** — creates branches, commits, and opens PRs
- **Deployment pipeline** — pushes trigger GitHub Actions → Vercel deploy
- **Telegram interface** — manage the entire dev cycle from your phone
- **OCI hosting** — runs 24/7 on Oracle Cloud free tier

### Tech Stack

- Python (Telegram bot framework)
- Google Gemini 2.5 Flash API
- Oracle Cloud Infrastructure (Compute)
- GitHub Actions for CI/CD
- Vercel for deployment
