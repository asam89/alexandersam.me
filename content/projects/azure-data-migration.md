## Azure Data Migration

Enterprise data migration from legacy student information systems to Azure.

### Key Deliverables

- **Denormalized table design** optimized for SaaS consumption patterns
- **Event-driven architecture** enabling real-time data synchronization
- **ETL pipelines** handling complex data transformations at scale
- **SaaS enablement layer** allowing downstream applications to consume unified data

### Tech Stack

- Microsoft Azure (Storage, Functions, Service Bus)
- SQL Server with denormalized schemas
- ETL pipelines with error handling and retry logic
- Event-driven messaging for downstream consumers
