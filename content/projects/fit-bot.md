## Fit Bot

A personal fitness and nutrition tracking bot that uses AI to analyze meals from photos and provide personalized health nudges.

### Key Features

- **Photo-based meal logging** — snap a photo, get nutrition breakdown
- **AI nutrition analysis** with calorie and macro tracking
- **Meal type labels** and meal management (add/delete)
- **Personalized nudges** based on habits and goals
- **Multi-user web dashboard** with per-user Telegram connections
- **Diary page** for reviewing daily intake history

### Tech Stack

- Python
- Telegram Bot API
- AI-powered image recognition and nutrition analysis
- Docker for deployment
