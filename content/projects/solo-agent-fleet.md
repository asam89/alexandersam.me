## Solo Agent Fleet

Orchestrated collection of autonomous AI agents.

### Key Features

- **Multi-agent architecture** — each agent handles a specific domain
- **Centralized control plane** for monitoring and configuration
- **Budget tracking** — per-agent API spend management
- **Domain-specific agents** — career, marketing, finance, presence

### Tech Stack

- Python agent framework
- Claude API and Gemini for LLM tasks
- OCI for hosting
- GitHub Actions for CI/CD
