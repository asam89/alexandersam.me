## MyCareer Bot

AI-powered career automation and job application management.

### Key Features

- **Job analysis** — AI scores fit between your profile and job postings
- **Tailored materials** — generates customized resumes and cover letters per application
- **Application tracking** — pipeline from Saved → Applied → Interview → Offer
- **Auto-apply** — browser automation for submitting applications via Playwright
- **Budget management** — tracks AI API spend per agent

### Tech Stack

- FastAPI + Python
- Anthropic Claude API
- Playwright for browser automation
- SQLite for persistence
- Jinja2 + WeasyPrint for PDF generation
