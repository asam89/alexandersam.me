## Daily Briefing Platform

Real-time intelligence platform aggregating multiple data sources.

### Key Features

- **Market data** — stocks, crypto, and forex updates
- **News aggregation** — politics, tech, and world events
- **Telegram delivery** — daily briefings sent directly to your chat
- **Worker architecture** — distributed processing on OCI Ampere instances

### Tech Stack

- Python worker nodes
- Telegram Bot API
- Oracle Cloud Infrastructure (OCI)
- Multiple news and market data APIs
