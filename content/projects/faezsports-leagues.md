## FaezSports Leagues Platform

Full-stack sports management platform serving community organizations.

### Key Features

- **Multi-tenant architecture** with subdomain routing per league
- **Real-time game statistics** tracking with custom sport configurations
- **Tournament brackets** with automated winner advancement
- **Registration & payments** via Stripe integration
- **AI-powered league setup** using Claude API for config generation

### Tech Stack

- Next.js (App Router) with TypeScript
- Prisma ORM + PostgreSQL
- Stripe for payments
- Tailwind CSS
- Deployed on Vercel
