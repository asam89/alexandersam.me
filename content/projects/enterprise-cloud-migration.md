## Enterprise Cloud Migration

Multi-year cloud migration program for financial institutions.

### Key Deliverables

- **Workload migration** from on-premises data centers to Azure
- **Zero Trust architecture** implementation across hybrid environments
- **Network redesign** with hub-spoke topology and private endpoints
- **IAM modernization** with Conditional Access and Privileged Identity Management

### Tech Stack

- Microsoft Azure (Landing Zones, Hub-Spoke)
- Azure Networking (ExpressRoute, Private Link, NSGs)
- Identity & Access Management (Entra ID, PIM, Conditional Access)
- Zero Trust security model
